## 📌 問題提起

犬や猫の写真を撮るとき、「せっかく撮ろうとしてもカメラを見てくれない…」という経験をしたことはありませんか？

本アプリは、ペットの注意を引きつける「音」を活用し、撮影の瞬間にペットがしっかりカメラを見てくれるようサポートします。

## 👤 ターゲットユーザー<br>

•	犬・猫などペットを飼っている飼い主<br>
•	ペットとの思い出を写真で残したい人<br>
•	写真をもっと楽しく、便利に管理したい人<br>

## ✨ 主な機能

#### 🎵 1. 注意を引くサウンド × フィルター＆スタンプ<br>
•	ペットの好奇心を刺激するサウンドを再生して視線をカメラへ誘導<br>
•	ペット写真に特化した可愛いフィルター<br>
•	写真を楽しくアレンジできるキュートなスタンプ機能<br>

#### 🗺️ 2. 地図連携フォトアルバム<br>
•	撮影時のGPS情報を自動取得し、写真に位置情報タグを付与<br>
•	ペットと訪れた場所を地図上で一覧でき、思い出を視覚的に管理<br>
•	時間・場所で写真をフィルタリングし、回想モードで振り返り可能<br>

#### 💳 3. ペット専用・個人名刺カード<br>
•	ペットの写真、名前、年齢、品種、性別、好みなどを登録<br>
•	可愛い背景やスタンプでカスタマイズ可能<br>
•	公園などで他の飼い主と交流するきっかけに<br>

#### 📡 4. MultipeerConnectivity による名刺交換<br>
•	近くのデバイスを自動検出し、ワンタップで名刺を交換<br>
•	過去に交換した相手の履歴を保存して、後から連絡しやすい<br>
•	飼い主同士の新しいコミュニティ体験を創出<br>

#### 📝 5. ホーム画面の気持ちメモ<br>
•	撮影後にその時の気持ちや小さなストーリーを記録<br>
•	写真一枚一枚に“思い出の文章”が加わり、より大切な記録に<br>

#### 🧭 機能一覧（まとめ）<br>
•	ペットの注意を引くサウンド<br>
•	専用フィルター & スタンプ<br>
•	自動位置情報タグ<br>
•	地図で思い出を管理 & 回想<br>
•	ペット名刺（カスタムデザイン）<br>
•	MultipeerConnectivity 名刺交換<br>
•	撮影時の“気持ちメモ”<br>
